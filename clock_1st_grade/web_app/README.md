このフォルダには時計学習用の簡易Webアプリがあります。

使い方:
1. `index.html` をブラウザで開きます（ローカルでそのまま開けます）。
2. 「出題をだす」を押すと、5分刻みの時刻が表示されます。
3. 画面上の長針／短針をドラッグして、出題された時刻に合わせます。
4. 「判定」を押して、正誤を確認します。

注意:
- タブレットやタッチ画面でも動作するように `pointer` イベントを使用しています。
- 本プロトタイプは簡易版です。必要なら「針のスナップ」「音声読み上げ」「アクセシビリティ対応」などを追加できます。
 - 正解時に短い効果音（組み込みのWebAudio）とコンフェッティ表示が出ます。
 - 効果音は外部ファイル不要でブラウザ側で鳴らします。音を出したくない場合は `script.js` の `playSuccessSound` 呼び出しをコメントアウトしてください。
 - 正解時に短い効果音（組み込みのWebAudio）とコンフェッティ表示が出ます。
 - 外部効果音ファイルを使いたい場合は、以下のいずれかのパスに短い効果音ファイルを置くと自動で優先再生されます（ファイルが見つからない場合は組み込み音を使います）。

  1. `web_app/sounds/success.mp3`
  2. `web_app/sounds/success.wav`
  3. `web_app/success.mp3`
  4. `web_app/success.wav`

 - 音声ファイルは短め（300〜800ms 推奨）で、授業用にライセンスが許可された効果音を使用してください。無料で使える素材例：
  - Freesound（要ライセンス確認）
  - 効果音ラボ（日本語サイト）

 - 追加した外部ファイルを使わない場合は何も置かなくて構いません。
